# SWRT (Snow Rut Time)

|instruction| arg1 | arg2 | arg3 |
|-----------|------|------|------|
|    load   | reg  | imm  | N/A  |
|    add    | reg  | reg  | reg  |
|    sub    | reg  | reg  | reg  |
|    div    | reg  | reg  | reg  |
|    mul    | reg  | reg  | reg  |
|    eq     | reg  | reg  | N/A  |
|    neq    | reg  | reg  | N/A  |
|    gt     | reg  | reg  | N/A  |
|    inc    | reg  | N/A  | N/A  |
|    dec    | reg  | N/A  | N/A  |
|    jmp    |      label name    |
|    jeq    |      label name    |
|    jne    |      label name    |
|    prts   |      label name    |
|    hlt    |        N/A         |
|    nop    |        N/A         |
|    ige    |        N/A         |
